=====Employe managemnet CRUD APPLICATION ==========
A fullstack employee management system built using React js , Node JS exress Js , Mysql , JWT, Authenticatioin and axios.

-=====Features 
->user authentication & Login
->JWT Authentication
->Protected Route
->Add Employe
->View Employee
->Update Employee
->Delete Employee
->search Employee
->loading & Error handling
->MYSQL Database Integration



Used Technology 
->React JS
->node JS
->Express JS
->MySql
->JWT
->Bcrypt
->JWT
